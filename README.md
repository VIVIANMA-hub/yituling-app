# 一图灵 / 医图灵（YiTuLing）· 参赛展示版

> 面向基层医务工作者的**非诊断类**效率工具 + 经验共创社区。

本仓库是**参赛评审与技术展示用的公开版本**，包含 App 的前端界面、交互与公共工具代码。以下内容**不随本仓库发布**：

## 产品简介

一图灵是一款帮助放射/医技等基层医务工作者提升日常效率的工具型 App，核心做四件事：

- **智能排班**：拍照 OCR 解析排班表，自动生成带水印的分享图，支持在线响应与调班协同。
- **极速脱敏**：框选敏感区后在本地 Canvas 真实抹除像素，确保影像合规脱敏后才能展示或分享。
- **医保收费 + 体位避坑词典**：沉淀一线规范，降低差错与退费风险。
- **经验共创社区**：以"署名 + 激励"方式沉淀医务一线 UGC 经验。

## 合规红线

本项目坚持**不做影像自动分析、疾病判断、辅助诊断或用药分诊**；用户上传的影像必须先脱敏才能显示或分享。

## 技术栈

| 层 | 技术 |
|----|------|
| 框架 | uni-app（Vue 3 语法，HBuilderX 开发） |
| 目标平台 | iOS App、Android App、微信小程序（H5 备用） |
| 云端 | uniCloud（阿里云节点，云函数不在本仓库） |
| 状态管理 | Vuex |
| UI | 自定义 Apple 风格 + glassmorphism |

> OCR / AI 解析、登录短信、第三方登录等能力均通过云函数调用；所有密钥存于云端，**不进客户端、不进仓库**。
